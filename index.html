Quick Start

Panduan cepat untuk mulai menggunakan TopUpKu dalam 5 menit

1

Daftar Akun

Buat akun developer di dashboard.topupku.id dan verifikasi email Anda. Proses pendaftaran gratis dan hanya membutuhkan waktu 1 menit.

2

Buat API Key

Masuk ke dashboard dan buat API Key baru di menu Settings > API Keys. Simpan key Anda dengan aman dan jangan bagikan ke siapapun.

3

Integrasikan API

Gunakan API Key untuk melakukan autentikasi dan mulai membuat permintaan ke endpoint kami. Lihat contoh kode di bawah ini.

4

Test & Deploy

Gunakan sandbox environment untuk testing. Setelah siap, switch ke production mode dan mulai melayani pelanggan Anda.

JavaScript — Quick Start
const TopUpKu = require('@topupku/sdk');
// Inisialisasi client
const client = new TopUpKu({
  apiKey: 'your_api_key_here',
  environment: 'production' // atau 'sandbox'
});
// Cek saldo
const balance = await client.getBalance();
console.log(`Saldo: Rp ${balance.amount}`);

Info Sandbox

Gunakan environment sandbox untuk testing. Semua transaksi di sandbox menggunakan data dummy dan tidak memerlukan pembayaran sungguhan.
