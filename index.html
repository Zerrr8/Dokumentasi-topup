Quick Start

Panduan cepat untuk mulai menggunakan Upcointop dalam 5 menit

1

Daftar Akun

Buat akun developer di upcointop.dev (Progres) dan verifikasi email Anda. Proses pendaftaran gratis dan hanya membutuhkan waktu 1 menit.

2

Buat API Key

Masuk ke dashboard dan buat API Key baru di menu Settings > API Keys. Simpan key Anda dengan aman dan jangan bagikan ke siapapun.

3

Integrasikan API

Gunakan API Key untuk melakukan autentikasi dan mulai membuat permintaan ke endpoint kami. Lihat contoh kode di bawah ini.

4

Test & Deploy

Gunakan sandbox environment untuk testing. Setelah siap, switch ke production mode dan mulai melayani pelanggan Anda.

JavaScript — Quick Start 
const upcointop = require('@upcointop/sdk');

// Inisialisasi client
const client = new upcointop({
  apiKey: 'your_api_key_here',
  environment: 'production' // atau 'sandbox'
});

// Cek saldo
const balance = await client.getBalance();
console.log(`Saldo: Rp ${balance.amount}`);

// Top up game
const order = await client.createOrder({
  game: 'mobile-legends',
  userId: '123456789',
  zoneId: '1234',
  item: '86-diamonds',
  payment: 'qris'
});

console.log('Order berhasil:', order.invoiceUrl);

Info Sandbox

Gunakan environment sandbox untuk testing. Semua transaksi di sandbox menggunakan data dummy dan tidak memerlukan pembayaran sungguhan.

Proses Top Up

Alur proses top up dari awal hingga selesai

1

Pilih Game & Item

Pilih game yang ingin di-top up, lalu pilih nominal item (diamond, UC, dll) yang diinginkan.

2

Masukkan User ID

Masukkan User ID dan Zone ID (jika diperlukan) dari akun game Anda. Pastikan data yang dimasukkan benar.

3

Pilih Pembayaran

Pilih metode pembayaran yang tersedia: QRIS, Transfer Bank, E-Wallet, atau Virtual Account.

4

Bayar & Konfirmasi

Lakukan pembayaran sesuai instruksi. Setelah pembayaran dikonfirmasi, item akan otomatis masuk ke akun game Anda.

5

Selesai! 🎉

Item berhasil ditambahkan ke akun game. Anda akan menerima notifikasi via email dan webhook (jika menggunakan API).

Penting: Verifikasi User ID

Selalu verifikasi User ID sebelum melakukan pembayaran. Kami tidak bertanggung jawab atas kesalahan pengiriman akibat User ID yang salah.

Autentikasi API

Cara mengautentikasi permintaan API Anda

Semua permintaan API memerlukan autentikasi menggunakan API Key. Sertakan API Key Anda di header setiap permintaan.

Header Authentication 
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
X-Client-Version: 3.0.0

Keamanan API Key

Jangan pernah membagikan API Key Anda atau menyimpannya di kode client-side (frontend). Gunakan server-side untuk semua permintaan API.

FAQ

Pertanyaan yang sering ditanyakan

Berapa lama proses top up?
Proses top up biasanya membutuhkan waktu kurang dari 30 detik setelah pembayaran dikonfirmasi. Untuk beberapa game, proses bisa memakan waktu hingga 5 menit tergantung server game.
Apakah aman melakukan top up di Upcointop?
Ya, sangat aman. Kami menggunakan sistem pembayaran yang terenkripsi dan bekerja sama langsung dengan publisher game. Semua transaksi diproses secara legal dan aman.
Bagaimana jika top up tidak berhasil?
Jika top up tidak berhasil, dana Anda akan otomatis dikembalikan (refund) dalam 1x24 jam. Anda juga bisa menghubungi tim support kami untuk bantuan lebih cepat.
Apakah bisa menjadi reseller?
Tentu! Kami menyediakan program reseller dengan harga khusus. Cukup daftar melalui dashboard, buat API Key, dan mulai berjualan. Semakin banyak transaksi, semakin besar diskon yang Anda dapatkan.
Aplikasi premium apa saja yang tersedia?
Kami menyediakan langganan untuk Spotify Premium, Netflix, YouTube Premium, Canva Pro, Google One, Disney+ Hotstar, dan masih banyak lagi. Cek bagian Aplikasi Premium untuk daftar lengkap.
