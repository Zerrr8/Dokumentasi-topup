Proses Top Up

Alur proses top up dari awal hingga selesai

1

Pilih Game & Item

Pilih game yang ingin di-top up, lalu pilih nominal item (diamond, UC, dll) yang diinginkan.

2

Masukkan User ID

Masukkan User ID dan Zone ID (jika diperlukan) dari akun game Anda. Pastikan data yang dimasukkan benar.

3

Pilih Pembayaran

Pilih metode pembayaran yang tersedia: QRIS, Transfer Bank, E-Wallet, atau Virtual Account.

4

Bayar & Konfirmasi

Lakukan pembayaran sesuai instruksi. Setelah pembayaran dikonfirmasi, item akan otomatis masuk ke akun game Anda.

5

Selesai! đźŽ‰

Item berhasil ditambahkan ke akun game. Anda akan menerima notifikasi via email dan webhook (jika menggunakan API).

Penting: Verifikasi User ID

Selalu verifikasi User ID sebelum melakukan pembayaran. Kami tidak bertanggung jawab atas kesalahan pengiriman akibat User ID yang salah.
